# 🚀 DevOps CI/CD Pipeline for Food Ordering Web App 🍔

## 📌 Project Overview

This project demonstrates a complete **CI/CD pipeline** for a **Food Ordering Web Application** using DevOps tools like Jenkins and Docker.

Whenever code is pushed to GitHub, the application is automatically:
- Built
- Tested
- Dockerized
- Deployed as a running container

---

## 🌐 Application Features

🍔 Food Ordering UI (Swiggy-like)  
🔍 Search functionality  
📂 Category filtering (Burger, Pizza, Pasta)  
🛒 Add to Cart  
❌ Remove from Cart  
💰 Total price calculation  
🎨 Responsive UI with modern design  

---

## ⚙️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js  
- **CI/CD Tool:** Jenkins  
- **Containerization:** Docker  
- **Version Control:** Git & GitHub
